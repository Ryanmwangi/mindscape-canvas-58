# HisMind Creatives Website

A responsive brand and digital strategy website for HisMind Creatives, built with TanStack Start, React, Vite, Tailwind CSS, and Radix UI components.

The site presents the studio's services, brand story, journal/blog content, and contact calls to action. It includes server-side rendering through TanStack Start and a Cloudflare-oriented server entry with a branded error page wrapper.

## Tech Stack

- React 19
- TanStack Start and TanStack Router
- Vite
- TypeScript
- Tailwind CSS 4
- Radix UI primitives
- Lucide React icons
- Cloudflare Workers configuration through Wrangler

## Requirements

Install one of these package managers:

- Bun, recommended because this repo includes `bun.lock`
- Node.js with npm

## Getting Started

Clone the project, then install dependencies:

```bash
bun install
```

Start the development server:

```bash
bun run dev
```

Vite will print the local URL in your terminal, usually:

```text
http://localhost:5173
```

Open that URL in your browser to view the project.

## Using npm Instead

If you prefer npm, install and run with:

```bash
npm install
npm run dev
```

## Project Structure

```text
src/
  assets/              Images used by the homepage and blog cards
  components/          Shared site components and UI primitives
  hooks/               Reusable React hooks
  lib/                 Error handling and utility helpers
  routes/              TanStack Router pages
  router.tsx           Router setup
  server.ts            Cloudflare/TanStack Start server wrapper
  start.ts             TanStack Start middleware setup
  styles.css           Global styles and Tailwind CSS
public/                Static public assets
vite.config.ts         Vite and TanStack Start configuration
wrangler.jsonc         Cloudflare Workers configuration
```

## Deployment Notes

This project includes `wrangler.jsonc` and a custom `src/server.ts` entry for Cloudflare-oriented deployment. Build the app first:

```bash
bun run build
```

Then deploy using your Cloudflare/Wrangler workflow.

If Wrangler is not installed globally, add it or run it through your package manager before deploying.

## Development Notes

- The route tree is generated by TanStack Router in `src/routeTree.gen.ts`.
- The Vite config uses `@lovable.dev/vite-tanstack-config`, which already registers the main TanStack, React, Tailwind, Cloudflare, and path alias plugins.
- Avoid manually adding those same plugins again in `vite.config.ts`, because duplicate plugin registration can break the app.
